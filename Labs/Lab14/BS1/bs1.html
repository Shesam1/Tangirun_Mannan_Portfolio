HTML Training

HTML 5 is a hyper text markup language that is used for structuring and presenting content on the World Wide Web. You will be learning the fifth and current version of the HTML standard.

CSS  Training

Css is a cascading style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and you will be learning CSS 3.

jQuery Training 

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is a free and open source software and has syntax is designed to make it easier to navigate a document, select DOM elements, create animations, handle events, and develop Ajax applications. You will be learning this as well since it is very important to learn especially alongside Javascript.

Bootstrap Training 

Bootstrap is a free and open-source front-end library for designing websites and web applications. It contains HTML and CSS based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions. You will be learning this in ET 710 alongside HTML, CSS and jQuery.

Requirements

In order to be able to participate in the program, you must have: A copy of your passport or state ID, a copy of your Social Security Card, a resume, A Highschool diploma or equivalent.

If you are not a U.S. Citizen, then you must have: a permanent Resident Card or Alien Registration Receipt Card or Employment Authorization Document Card.
